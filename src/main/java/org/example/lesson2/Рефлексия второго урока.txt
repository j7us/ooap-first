В случае метода right() выставлено не то предусловие, возомжно из-за копипаста предусловий между методами,
предусловием метода должно быть наличие элемента справа от курсора.

Для метода remove_all написано предусловие, что список не должен быть пустым. Данное предусловие может
быть ошибочно, т.к наличие элементов в списке не влияет на выполнение команды удаления.

Не добавлены методы получения статусов выполнения.
